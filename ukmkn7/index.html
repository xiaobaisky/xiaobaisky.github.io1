butterfly主题配置（一）

发表于2020-06-13|更新于2021-03-15|oc
|字数总计:1.2k|阅读时长:4分钟|阅读量:

准备工作：
node.js > git

  • npm install hexo-cli -g
  • hexo init blog
  • cd blog
  • npm install
  • hexo server

一、Page Front-matter

寫法 解釋
title 【必需】頁面標題
date 【必需】頁面創建時間
type 【必需】標籤、分類和友情鏈接三個頁面需要配置
description 【可選】頁面描述
keywords 【可選】頁面關鍵字
comments 【可選】顯示頁面評論模塊 (默認 true)
top_img 【可選】頁面頂部圖片
mathjax 【可選】顯示 mathjax (當設置 mathjax 的 per_page: false 時，才需要配置，默認 false)
katex 【可選】顯示 katex (當設置 katex 的 per_page: false 時，才需要配置，默認 false)
aside 【可選】顯示側邊欄 (默認 true)

二、Post Front-matter

寫法 解釋
title 【必需】文章標題
date 【必需】文章創建時間
tags 【可選】文章標籤
categories 【可選】文章分類
cover 【可選】文章縮略圖 (如果沒有設置 top_img, 文章頁頂部將顯示縮略圖，可設為 false / 圖片地址 / 留空)
description 【可選】文章描述
keywords 【可選】文章关键字
comments 【可選】顯示文章評論模塊 (默認 true)
toc 【可選】顯示文章 TOC (默認為設置中 toc 的 enable 配置)
toc_number 【可選】顯示 toc_number (默認為設置中 toc 的 number 配置)
top_img 【可選】文章頂部圖片
mathjax 【可選】顯示 mathjax (當設置 mathjax 的 per_page: false 時，才需要配置，默認 false)
katex 【可選】顯示 katex (當設置 katex 的 per_page: false 時，才需要配置，默認 false
copyright 【可選】顯示文章版權模塊 (默認 true)

音樂界面使用了插件 hexo-tag-aplayer。
使用方法請參考插件的文檔。

音樂頁面只是普通的 page 頁，按普通頁面操作生成就行。

2、打开 C:\Hexo-Blog\themes\Butterfly\layout\includes\head.pug

尾部添加 include ./third-party/aplayer.pug
